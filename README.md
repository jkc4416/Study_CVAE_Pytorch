# pytorch-mnist-CVAE
Conditional Variational AutoEncoder on the MNIST data set using the PyTroch

## Dependencies
- PyTorch
- torchvision
- numpy

## Results
Learned MNIST manifold with a condition of label (from 0 to 9)
